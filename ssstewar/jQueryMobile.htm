jQuery Mobile: This framework is used for skinning webpages to look like mobile applications. For my topic on jQuery Mobile, I will be showing how to setup pages & link those pages together.

 

·       First you need to set up an HTML file with your normal five tags(DOCTYPE, HTML, head, title, & body).

·       We need to link a couple files so head to jquerymobile.com, click on downloads then scroll down to “CDN hosted files.” Copy and paste these 3 links into your HTML Page inside your head section. The first one is all the CSS you are going to need. The second one is linking in the jQuery framework. And the third one is linking in the jQuery mobile framework.

·       jQuery mobile has a few default div’s we need to set up for any basic page.

o   The first one is: <div data-role="page"> It is actually a HTML5 attribute but is something that is built into jQuery mobile itself to set this div up correctly. This piece of code assigns a main page to our application. We can have several all of these in our application for multiple pages but every page has to be surrounded by this div tag.

o   Within this page we have to have: <div data-role="header"> this defines the header of your div. I am going to call my header “My first page”.

o   We also need two more more of these div tags, the next one is: <div data-role="main" class="ui-content"> this piece of code takes care some of the styling around the div with some padding, inside of this div I am going to have an h1 that says “Hello World” then I am going to add a paragraph that says “This is my first page”.

o   The last div that we need is: <div data-role="footer"> and this defines the footer section of our application. In this footer I am going to add an h2 with copyright info with your name and the year 2015.

 

·       Lets now add a second page.

·       Copy and paste the div sections you just made and change the corresponding names/titles. For example: “My first page” à “My second page”, “Welcome to my first page” à “Welcome to my second page”

·       You can now go ahead and run it but you’ll come to find out that we can only see our first page. I am now going to go ahead and show you how to link those two pages.

·       This is fairly simple. To start out we have to be able to differentiate between the two pages so ad an ID tag to each of them, I made mine “Page1” & “Page2”

·       To link the two pages together, all we need is an anchor tag. Add this under where you say “Welcome to my first/second page”. So for page one you will need to add a tag to link to Page two and for your href all you need is a hashtag and the id of your page. For example: <a href="#Page2">Page 2</a>

·       Do this to your second div section for page to link to your page 2 so we can link back from page 2 to page 1. For example: <a href="#Page1">Page 1</a>

·       Go ahead and run it! You should be able to navigate between both pages. I recommend you minimize your window to make it look like its on a mobile device.
