Saginaw Valley State University (Fall 2015)

Prof. George Corser's links to course templates.

Exercises

  1. Exercise 1: Create an HTML file using 20 tags from w3schools. Files must follow the form: .html. Files must be pushed to github. Due date: Sept. 9, 2015.
  2. Exercise 2: Create a CSS file using 3 selectors and 20 attibutes from w3schools. The CSS file must format the HTML file made in Exercise 1. Files must follow the form: .css. Files must be pushed to github. Due date: Sept. 14, 2015.
  3. Exercise 3: Create a JS file with 5 functions. The JS file must be called from the HTML file made in Exercise 1. Files must follow the form: .js. Files must be pushed to github. Due date: Sept. 21, 2015.
