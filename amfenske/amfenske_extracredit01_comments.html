Weebly vs. Bootstrap

1. Ease of Access.

Weebly - 10 Bootstrap - 5

Weebly - I just had to Google Weebly and it was the top link.

Bootstrap - If I Google Bootstrap, the top link is for getBootstrap, not startBootstrap.

2. Easy to Make Changes/Updates.

Weebly - 8 Bootstrap - 8

Weebly - Has buttons for easy drag and drop, and text boxes to update information.

Bootstrap - I have to edit the actual code myself, but I can do whatever I want and put it whereever I want.



3. Free to download/use.

Weebly - 10 Bootstrap - 10

Weebly - Free to use.

Bootstrap - Free to download.



4. Free to publish.

Weebly - 8 Bootstrap - 1

Weebly - Free to an extent.

Bootstrap - Need to get a domain name.



5. Lots of options.

Weebly - 7 Bootstrap - 10

Weebly - Uses templates and has limited options.

Bootstrap - Can edit however I want and make it look exactly how I want.



6. Easy to find help.

Weebly - 8 Bootstrap - 10

Weebly - Didn't see an option to ask someone a question.

Bootstrap - Just Google anything you need and throw it in your code.



7. Dependable Information.

Weebly - 10 Bootstrap - 10

I completely trust the information/help/code provided by both Weebly and Bootstrap.



8. Easy to read source information.

Weebly - 5 Bootstrap - 8

Weebly - Hard to decipher which parts of the code are doing what.

Bootstrap - Fairly easy to read because I made changes and know what things do.



9. Browsing Speed.

Weebly - 6 Bootstrap - 9

Weebly - Pages takes several seconds to load/open.

Bootstrap - Pages load immediately.



10. Professional Looking.

Weebly - 7 Bootstrap - 9

Weebly - Difficult to make specific changes to streamline site appearances.

Bootstrap - Easier to make specific changes to make the page look professional



Overall Preference

Overall, I like Bootstrap better because I can make the changes I want when I want without any limitations on options.
